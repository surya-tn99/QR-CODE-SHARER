A simple, secure, and efficient tool to verify links, encode them into a safe format, and generate QR codes for sharing. It ensures that the destination link is valid before generating the code.

## How to use
1.  Enter your destination link in the input field.
2.  Click **Generate QR Code**.
3.  Scan the generated QR code to be safely redirected to your link.
